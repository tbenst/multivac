## dev mode:
run in separate terminals
```
yarn spago build --watch
yarn parcel assets/*.html
```
## useful commands
```
yarn spago serve
```

## init project
```
yarn add --dev purescript spago parcel-bundler
yarn spago init
yarn spago install halogen
mkdir -p dist
```
